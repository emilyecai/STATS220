Introduction

  • I decided to focus on exploring the learning preferences of people learning different languages. I was curious to see how interested people were in learning a new language, what languages people currently know, if people were making a conscious effort to learn new languages, etc.

  • One of the guidelines I considered when creating this Google Form was Guideline 1: Communicate your purpose. I considered this while designing my form by adding an introduction explaining the purpose of the data collection, including who it will be shared with.

  • The categorical data that was generated using my questions with check boxes (fixed options) means the Google sheet data collected using my form is not strictly tidy data because the data from each respondent are put into one cell in the Google sheet. If the respondent chooses multiple answers, the answers will all be inputted into a singular cell for each entity, separated by commas. This is not tidy data as there are multiple answers per cell.

Learning Reflection

  • An important idea I learnt from Module 2: Creating dynamic reporting systems is that we need to take measures in designing surveys for collecting data in a way that gives us the most useful and clean data in the first place possible, so that it will be easier when we wrangle and clean our data later. An example of doing this is to use regex in our forms to restrict the types of answers our respondents give us.

  • I demonstrated creativity in this project by:

  1. Using regex in my google form questions: I added an extra question in the form asking for the names of any language courses the student has taken at The University of Auckland. I added the regular expression [A-Z]{5,10}[0-9]{3} to ensure that respondents entered a word between 5 and 10 characters followed by a number of 3 digits as this is the format that the languages courses at UoA follow.
  2. Using skills/knowledge covered in Module 1 not asked for in this project such as: adding a CSS chunk to the index.Rmd file to alter the visual appearance of the file, changing properties such as text/background color and font imported from Google Fonts.
  • I am now curious about the different types of databases that data can be stored in. In class we have learnt about relational databases. I would like to learn more about the use of hierarchical/network databases and object-oriented/NoSQL databases and the differences between each one, as well as why we would choose to use one over another in different circumstances.
